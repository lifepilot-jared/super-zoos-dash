# Super Zoos Adventure v0.6 — The Living School

## Goal
Create one polished, five-minute iPad mission that makes Eli feel Peter and Judy are alive inside a believable Australian school world.

## Build principles
- Keep controls simple: swipe left/right, swipe up to jump, swipe down to duck, Shield.
- Preserve ASD/ADHD-friendly pacing: clear signs, no harsh failure, predictable tutorial sequence.
- Use original Super Zoos assets only.
- Keep GitHub Pages deployment stable.
- Add one system at a time and test before continuing.

## Locked character states
Each playable hero supports:
- normal-front
- normal-side
- normal-back
- super-front
- super-side
- super-back
- aura-front
- aura-side
- aura-back

Gameplay state meaning:
- Normal: ordinary plush form.
- Super: transformed hero suit.
- Aura: fully powered temporary mode.

## Sprint order
1. Foundation audit and modular config.
2. Character asset manifest with safe fallbacks.
3. Wider 2.5D school camera and environment layers.
4. Running animation system and hero acknowledgement animation.
5. Terrain system: hill, balance beam, low obstacle.
6. Duck/slide control and collision rules.
7. Dr Winnie scripted encounter.
8. Audio verification and custom voice-clip slots.
9. iPad play test and tuning.
10. Merge only after build and playability checks pass.

## First playable v0.6 mission
- Peter greets Eli.
- Start at the school oval.
- Collect stars and hero badges.
- Run through a gentle bend.
- Run up a small hill.
- Cross a balance beam.
- Duck under one low obstacle.
- Dr Winnie triggers one obvious school obstacle event.
- Peter celebrates a strong run with eye contact, flex pose and trumpet sound.

## Acceptance criteria
- Production build passes.
- Existing v0.5 game remains untouched on main until approval.
- Rear-view asset fallback works even before PNG uploads are complete.
- Perspective is wider and less tunnel-like.
- Ground and side scenery visibly move.
- Jump, duck, lane movement and Shield are reliable on iPad Safari.
- Sound test works after one user tap.
- Dr Winnie event is visually obvious.
- Peter/Judy normal, super and aura states are distinguishable.
