# Super Zoos Dash MVP Test Checklist

Use this checklist after GitHub Pages deploys the placeholder MVP.

## Open the game

- [ ] Page loads on iPad Safari.
- [ ] The screen fits without awkward scrolling.
- [ ] Start screen is easy to understand.

## Core controls

- [ ] Start Run begins the game.
- [ ] Tapping the oval makes Peter jump.
- [ ] Jump does not double-trigger wildly.
- [ ] Shield button is large and easy to press.
- [ ] Pause button works.
- [ ] Resume works.
- [ ] Restart works.

## Gameplay feel

- [ ] Obstacles feel fair, not instant or overwhelming.
- [ ] Meteor, ice, and lightning are visually distinct.
- [ ] Star gems are collectable.
- [ ] Hero badge transforms Peter into Super Peter.
- [ ] Shield clears obstacles without losing a heart.
- [ ] Hearts reduce when Peter is hit.
- [ ] End screen says “Great run, Super Peter!” and shows score/best.

## Accessibility feel

- [ ] No flashing effects feel harsh.
- [ ] Calm Mode is noticeably gentler.
- [ ] Text is readable.
- [ ] The game feels playful, not scary.

## Next polish after test

- Tune jump height and gravity.
- Tune obstacle spawn spacing.
- Tune shield duration/cooldown.
- Add clean Peter and Super Peter side images.
- Add short Peter trumpet and shield sounds.
